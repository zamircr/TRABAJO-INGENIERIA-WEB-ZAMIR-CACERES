## Accesibilidad

### Contraste de Colores
- Ratio mínimo WCAG AA: 4.5:1 para texto normal
- Ratio mínimo WCAG AAA: 7:1 para texto normal

### Espaciado
- Tamaño mínimo de toque: 44x44px
- Espaciado entre elementos interactivos: mínimo 8px

### Fuentes
- Tamaño mínimo de fuente: 16px para texto legible
- Altura de línea mínima: 1.5
